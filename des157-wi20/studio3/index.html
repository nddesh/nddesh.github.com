How to Play

Attempt to beat the game by getting a count as close to 21 as possible, without going over 21. The game has 3 rounds. Use the "Hit Me" button to draw a card. If you go over 21, you lose and the game is over! You can quit the game at any time by pressing the "Quit Game" button. Have fun!
